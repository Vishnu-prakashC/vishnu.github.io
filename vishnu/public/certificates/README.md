# Certificate images

Place your certificate or issuer logos here for the Certification Vault.

Expected filenames (from `src/data/certifications.js`):

- `aws.png` — AWS Developer Associate
- `react.png` — React Advanced Developer
- `gcp.png` — Google Cloud Professional Developer

Use square or landscape images; the UI scales them. If a file is missing, the section shows the issuer’s first letter as a fallback.
